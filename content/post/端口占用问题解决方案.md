# 问题原因  
因为8080端口被占用  

# 解决方案一  
> * 1. 打开cmd
> * 2. 输入：netstat -ano
> * 3. 查看所有端口信息，找到端口 8080，以及对应的 PID，如：  
![](/img/vue_eeor_1.png)
> * 4. 输入：tskill PID 即可杀死进程  

# 解决方案二
> * 1. 打开cmd
> * 2. 输入：netstat -ano
> * 3. 查看所有端口信息，找到端口 8080，以及对应的 PID
> * 4. 输入：tasklist，查看所有任务列表，找到PID 对应的任务，如：  
![](/img/vue_error_2.png)
> * 5. 输入：tskill PID 即可杀死进程；或者打开任务管理器，结束对应进程。
